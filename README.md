# 📄 Currículo Web - Lucas Martins Barreto

Este é o repositório do meu currículo online, desenvolvido com HTML5 e CSS3. O projeto foi estruturado de forma limpa, responsiva e moderna para apresentar minhas informações profissionais, acadêmicas e de contato.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização personalizada, layout responsivo (Mobile First) e design moderno.

## 📱 Funcionalidades

* **Design Responsivo:** Adaptável para dispositivos móveis, tablets e desktops.
* **Seções Organizadas:** Divisão clara entre "Sobre Mim", "Formação Acadêmica", "Experiência" e "Contato".
* **Clean UI:** Visual moderno utilizando tons de azul escuro e cinza claro para uma leitura agradável.

---
